## 8.深浅拷贝

> 什么是浅拷贝？如何实现浅拷贝？什么是深拷贝？如何实现深拷贝？

一般我们用 Object.assign 和扩展运算符实现浅拷贝

通常可以用 JSON.parse(JSON.stringify(Obj))实现深拷贝

但是该方法有局限性：

- 忽略函数
- 忽略 undefined
- 忽略 symbol
- 不能序列化函数
- 不能解决循环引用的对象

也可以使用 MessageChannel 实现深拷贝

该方法有局限性：

- 不能拷贝函数
