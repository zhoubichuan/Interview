## 7. css 选择器分类

* 基本的：

1.  id 选择器（id="name"）
2.  类选择器（class="name"）
3.  标签选择器（body，div，ul，li）
4.  全局选择器（\*）

* 复杂的

1.  组合选择器（.head .head_logo）
2.  后代选择器（#head .nav ul li 从父级到子孙集）
3.  群组选择器（div，span，img，{color：red}具有相同样式的标签分组显示）
4.  继承选择器
5.  伪类选择器（连接样式，a 元素的伪类）
6.  子选择器（div > p，带大于号）
7.  css 相邻兄弟选择器（h1 + p,带加号）
