## 1.this 指向

函数作为对象的方法被调用时：

this 指向该对象

```
var obj={
    fn:function(){
        console.log(this)
    }
}
obj.fn()
```

函数作为普通函数被调用时：

this 指向 window

```
function fn(){
    console.log(this)
}
fn()
```

函数作为构造器被调用时：this 指向返回的这个对象

```
function Fn(){
    this._fn=function(){
        console.log(this)
    }
}
var fn = new Fn()
fn._fn()
```

this 的隐式丢失：

this 指向 window

```
var obj={
    fn:function(){
        console.log(this)
    }
}
var fn=obj.fn
fn()
```

箭头函数:

箭头函数的 this 绑定看的是 this 所在函数定义在哪个对象下，就绑定哪个对象；如果有嵌套的情况，则 this 绑定到最近的一层对象上，this 的指向固定化，并不是因为箭头函数内部有绑定 this 的机制，实际原因是箭头函数根本没有自己的 this,导致内部的 this 就是外层代码块的 this。正是因为它没有 this，所以也就不能用作构造函数；


## 2.怎么改变 this 指向呢

* 1.使用箭头函数
* 2.在函数内部使用 that = this
* 3.使用 apply,call,bind
* 4.new 实例化一个对象
