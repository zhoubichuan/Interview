## 介绍下 Set、Map、WeakSet 和 WeakMap 的区别

- Set
  - 成员的值都是唯一的，没有重复的值
  - 两个属性 constructor,size
  - 可以遍历
  - 四个方法：add,delete,has,clear
- weakSet
  - 不重复的值的集合
  - 成员只能是对象，而不能是其他类型的值
  - 没有 size 属性，不能遍历
  - 三个方法 add,delete,has
- Map
  - javascript 的对象，本质上是键值对的集合
  - 但是传统上只能用字符串当做键
  - 可以遍历，方法很多，可以干根各种数据格式转换
- weakMap
  - 只接受对象作为键名
  - 键名指向的对象，不计入垃圾回收机制
  - 不能遍历，方法同 get,set,has,delete
